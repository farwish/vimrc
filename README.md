# vimrc


Files:

```
.vimrc   
.gitconfig   
```

Use .vimrc:

`git clone https://github.com/VundleVim/Vundle.vim.git ~/.vim/bundle/Vundle.vim`  
`git clone https://github.com/farwish/vimrc.git ~/vimrc && cp ~/vimrc/.vimrc ~/`  
`vim`  
`:PluginInstall`  

Use .gitconfig:  

`cp ~/vimrc/.gitconfig ~/`
